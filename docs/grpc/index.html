TemporalAuth

TemporalAuth implements Temporal's authentication and account management APIs

Method NameRequest TypeResponse TypeDescription
Register RegisterReq User

Register facilitates user creation.
Recover RecoverReq Empty

Recover facilitates account recovery.
Login Credentials Token

Login accepts credentials and returns a token for use with further requests.
Account Empty User

Account returns the account associated with an authenticated request.
Update UpdateReq User

Update facilitates modification of the account associated with an authenticated request.
Refresh Empty Token

Refresh provides a refreshed token associated with an authenticated request.

TemporalStore

TemporalStore implements Temporal's storage APIs

Method NameRequest TypeResponse TypeDescription
Upload UploadReq stream Object

Upload uploads a stream of blobs - it accepts files and directories. Blobs larger than 5mb should use this API
UploadBlob UploadReq Object

UploadBlob allows the upload of a single blob - if it is too large, an error will be returned
Download DownloadReq Blob stream

Download retrieves an object as a stream of blobs. Blobs larger than 5mb should use this API
DownloadBlob DownloadReq Blob

DownloadBlob allows the download of a single blob - if it is too large, an error will be returned
Pin Object Empty

Pin handles new pins and pin extensions
Stat Object ObjectStats

Stat retrieves details about an object
ListObjects ListObjectsReq ObjectList

ListObjects retrieves a list of the authenticated user's objects
Publish Event Empty

Publish publishes a message to the requested topic
Subscribe Topic Event stream

Subscribe subscribes to messages from the requested topic
Keys Empty KeyList

Keys returns the IPFS keys associated with an authenticated request
NewKey Key Empty

NewKey generates a new IPFS key associated with an authenticated request.
